The folder MarsRoversJava\src\marsrover contains the MarsRoverJavaMain.java, which is my solution to the Mars Rover problem, written in Java.

The folder MarsRoverJata\test contains the JUnit test-suite.

To run the program, compile the program using the command:

	javac MarsRoverJavaMain.java

Run the program by passing the absolute path to the input file to it:

	java MarsRoverJavaMain "C:\....\MarsRoverInputFile"
